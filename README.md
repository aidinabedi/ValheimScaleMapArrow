# Scale Map Arrow

A mod for Valheim that allows you to scale the size of the local player arrow on both the minimap and large map.

## Prerequisites

- Install a mod manager like [R2Modman](https://thunderstore.io/package/ebkr/r2modman/).
- Install [BepInExPack Valheim](https://thunderstore.io/c/valheim/p/denikson/BepInExPack_Valheim/) with mod manager.

## Install from Thunderstore

Install [ScaleMapArrow](https://thunderstore.io/c/valheim/p/Aidin/ScaleMapArrow/) with mod manager.

## Build from source

1. Install .NET SDK (Core 3.1 or later) from [Microsoft](https://dotnet.microsoft.com/en-us/download).
2. Download source code from [GitHub](https://github.com/aidinabedi/ValheimScaleMapArrow).
3. Build release with .NET CLI command: `dotnet build -c Release`
4. Import generated `ValheimScaleMapArrow.zip` as local mod using mod manager.
